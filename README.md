# Q&A

A Knowledge Base Theme for [Hexo].

- [Preview](http://chengkang.me/hexo-theme-q-a/)

## Installation

### Install

``` bash
$ git clone https://github.com/cheng-kang/hexo-theme-q-a.git themes/Q&A
```

### Enable

Modify `theme` setting in `_config.yml` to `Q&A`.

### Update

``` bash
cd themes/Q&A
git pull
```
